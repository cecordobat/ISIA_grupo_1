# Glosario

| Termino | Definicion |
|---|---|
| IBC | Ingreso Base de Cotizacion sobre el que se calculan aportes. |
| SMMLV | Salario Minimo Mensual Legal Vigente. |
| UVT | Unidad de Valor Tributario usada en retencion. |
| CIIU | Codigo de actividad economica usado para costos presuntos. |
| Costos presuntos | Porcentaje deducible del ingreso segun CIIU. |
| SGSSI | Sistema General de Seguridad Social Integral. |
| PILA | Planilla Integrada de Liquidacion de Aportes. |
| ARL | Riesgos laborales segun nivel de riesgo. |
| UGPP | Entidad que fiscaliza aportes y puede sancionar. |
| BEPS | Beneficios Economicos Periodicos del piso de proteccion social. |
| Retencion en la fuente | Anticipo de renta calculado sobre base depurada. |
| SnapshotNormativo | Foto de parametros legales vigentes al momento del calculo. |
| Append-only | Historico inmutable: no se edita ni se elimina. |
