# ISIA_grupo_1
Repositorio del proyecto final para el curso Ingeniería de Software Asistida por Inteligencia Artificial. Motor de liquidación de aportes a la seguridad social (SSSI) para independientes en Colombia. Sistema basado en reglas que automatiza el cálculo del IBC, aportes de ARL/Pensión y retenciones bajo la normatividad vigente (Res. 209/20)
